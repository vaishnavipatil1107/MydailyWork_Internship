# Number Guessing Game (Java)

## Description
This is a simple Java console-based Number Guessing Game.
The program generates a random number and allows the user to guess it.

## Features
- Random number generation (1 to 100)
- User input for guessing
- Feedback: Too High / Too Low / Correct
- Limited number of attempts
- Multiple rounds support
- Score tracking

## Technologies Used
- Java
- Eclipse IDE

## How to Run
1. Clone the repository
2. Open project in Eclipse
3. Run the main class
4. Follow console instructions
